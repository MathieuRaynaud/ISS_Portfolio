Self evaluation

The Innovative Smart Systems formation allowed me to acquire a lot of skills in the domain of the IoT.

First, I improved importantly my skills in terms of software development. I learned new programming languages like NodeJS, Angular, and Node-Red. I also learned to produce a REST API and web services to interact with an application.

Then, I learned to lead a project using the scrum method.

In terms of hardware development, I obtained a lot of useful information, that allows me to work with hardware components with less difficulties.

Finally, I would say that the ISS formation is a very skillful formation which brought me a lot of competencies in both hardware and software domains.
